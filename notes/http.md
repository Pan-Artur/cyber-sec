HTTP запит це запит, який робиться на сервер завдяки якійсь дії, для того щоб щось змінити або щось отримати на самому сайті, приклад: 

Request-line: GET /login HTTP/1.1 
Headers: 
  Host: ... 
  User-Agent: ... 
  Cookie: ... ... 
  Body(не завжди): ... 
 
HTTP відповідь, це відповідь на якийсь запит від користувача, який прийшов на сервер і тепер як раз таки відповідь відправляється йому, в ній може бути: 

Status-line: HTTP/1.1 200 OK 
Headers: 
  Content-type: ... 
  Set-Cookie: ... 
Body(не завжди): ...
